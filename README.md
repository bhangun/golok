# Golok 
Tools to scaffold apps

![golok Result][golok-result]

## Prerequisite
- NodeJS
- Flutter

## Install
```bash
npm i -g golok
```

## Usage
```bash
golok create yourModelFile.yaml
```

## Contribution

### Clone golok
```bash
git clone to/your/golok
```
### Link golok
```bash
npm link to/your/golok
```

### Unlink golok
```bash
npm unlink to/your/golok
```
